## Overview
An interactive **HR Analytics dashboard built in Power BI** to analyze workforce distribution, employee satisfaction, performance ratings, and training cost trends.

This project focuses on **clean data modeling, DAX measures, and business-ready visuals**, reflecting how analytics dashboards are delivered in real-world environments.

## Tech Stack
- **Power BI Desktop**
- **Data Source:** CSV (`data/HR.csv`)
- **Modeling:** Star-schema style relationships
- **DAX:** Measures for KPIs and aggregations

## Key Insights
- Workforce distribution by department and job type
- Employee satisfaction and performance rating breakdowns
- Training cost trends and internal vs external training analysis
- Demographic views to support diversity analysis

## How to Use
1. Install **Power BI Desktop**
2. Download or clone this repository
3. Open `powerbi/HR-analysis.pbix`
4. If prompted, update the dataset path to `data/HR.csv`

## Design Notes
- Measures are written using **DAX** for reusability
- Visuals are optimized for clarity and business users
- Filters and slicers enable interactive exploration
